# Octave

## Project Description

Octave turns your browser's tab bars into a playable musical instrument. Open the page in up to 8 tabs, run a quick calibration to confirm which physical tab is which, and each tab gets assigned a note. Switch between tabs with Ctrl/Cmd+1 through 8 to play notes. Future plan: A built-in rhythm-game mode can also feed you a song to play back by guiding you to the correct tab at the correct time.

## Features

- Tab calibration across up to 8 open windows
- Automatic tab tracking using `BroadcastChannel`
- Note assignment from C4 to C5
- Keyboard-based playback with Ctrl/Cmd+1 through 8 tab switching
- Audio feedback with a simple sine-wave synthesizer
- Visual flash feedback when a note is played

## Controls

- Open this page in multiple tabs (8 recommended). **Make sure they are the first 8 tabs on your browser!!!**
- Click `Start Calibration` to begin assigning notes
- Use `Open Another Tab` to create an additional tab quickly
- Switch tabs with Ctrl/Cmd+1 through 8, then return to your assigned tab to play its note
