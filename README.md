# Octave

## Project Description

Octave turns your browser's tab bars into a playable musical instrument. Open the page in up to 8 tabs, run a quick calibration to confirm which physical tab is which, and each tab gets assigned a note. Switch between tabs with Ctrl/Cmd+1 through 8 to play notes. A built-in rhythm-game mode can also feed you a song to play back by guiding you to the correct tab at the correct time.
